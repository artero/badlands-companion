# Badlands Companion

[Badlands Companion](https://artero.github.io/badlands-companion) es un proyecto web donde los miembros de "La Morada de Othos" reunimos diferentes ayudas para nuestras partidas de Ironsworn Badlands

## Setup y desarrollo

### Requeriments

- [Jenkyll Gem](https://jekyllrb.com/)

### Setup y arrancar el servidor

```
git clone https://github.com/artero/badlands-companion.git
cd badlands-companion
bundle install
Jekyll s -l
```
